#  Network Transmutation Circle

An interactive network topology visualization inspired by **Fullmetal Alchemist**. Visualizes a three-tier enterprise network as a mystical transmutation circle.

**[Live Demo](https://yourusername.github.io/FMA-TransmutationCircle-Network-Topology/)**

##  Features

- Interactive SVG visualization with sacred geometry patterns
- Filter by network layer (Core, Distribution, Access, Endpoint)
- Color-coded VLAN highlighting
- Click devices to view configurations
- Alchemical symbols for each device type

##  Getting Started

```bash
cd frontend
npm install
npm run dev
```

Open `http://localhost:5173/FMA-TransmutationCircle-Network-Topology/`

##  Build for Production

```bash
npm run build
```

The `dist` folder can be deployed to GitHub Pages or any static host.

##  Cisco Packet Tracer

A `.pkt` file is included in `documentation/` for use with Cisco Packet Tracer.
